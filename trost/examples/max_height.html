Max Height

Defines the maximum height of an element to contrainst its vertical span.

Syntax

Keyword

To specify max-height based on em, just add maxh-[*] class element.

If you want it based on percentage, just add maxh--pct-[*] class element.

[*] in this content represents a number in the max-height hierachy.
