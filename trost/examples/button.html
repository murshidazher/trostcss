Button

Button is an essential component in establishing a web and apps smooth conversational flow.

Shapes

Button Default

Button styles can be added directly to button elements or on anchor tag by using the btn class. 

                <button class="btn">...</button>

Button Block

                <button class="btn btn--block">...</button>

Button Pill

                <button class="btn btn--pill">...</button>

Button Square

                <button class="btn btn--square">...</button>

Button Cirle

                <button class="btn btn--circle">...</button>

States

Button Disbaled

Add disbaled attribute to make a button unusable and un-clickable. 

              <button class="btn" disabled>...</button>

Types

Labled Icon

Button can contain both a label and icon by adding btn--icon class to the button. 

                  <div class="btn btn--icon">
  <svg>...</svg>
  <span>Button</span>
</div>

Button Icon

Button can contain only icon by adding btn--icon and btn--square or add btn--circle class to determine the shape of the button. 

                  <div class="btn btn--icon btn--square">
  <svg>...</svg>
</div>
<div class="btn btn--icon btn--circle">
  <svg>...</svg>
</div>
