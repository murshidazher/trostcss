Character Width

Defines the number of character that should be present in a line.

Syntax

Definition

Use the chw--[*]-[^] class to specify the character width.

[*] indicates the font face mono, sans or serif.

[^] indicates the ideal number of characters.

Measured

Narrow

Use chw--sans-narrow to have approx. 48 sans characters per line.
