Max Width

Defines the maximum width of an element to contrainst its horizontal span.

Syntax

Keyword

To specify max-width based on em, just add maxw-[*] class element.

If you want it based on percentage, just add maxw--pct-[*] class element.

[*] in this content represents a number in the max-weight hierachy.
