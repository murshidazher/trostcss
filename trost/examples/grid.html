Grids

Grid is a layout tool for creating a responsive layout on the web which is baked right into a stylesheet.

Layout

Syntax

Layout has a horizontal row and vertical col-[span-amount] structure to it. 

                    <div class="row">
  <div class="col-12">
    ...
  </div>
</div>

System

Grid Break

If a row exceeds the 12-column per row layout, the system by default shifts the excess column to the next line vertically.

Col 6
Col 4
Col 4

Fluid Row

If you want to override the default 12-column per row layout system and want all columns in the same row without a break to next line, then use row--fluid and flex--wrap-nowrap classes.

Col 6
Col 8
Col 4
                  <div class="row row--fluid flex--wrap-nowrap">
  <div class="col-6">
    ...
  </div>
  <div class="col-8">
    ...
  </div>
  <div class="col-4">
    ...
  </div>
</div>

The 12-column layout grid system will ignore the content placed inside the grid if they have an absolute size hence if you want to conside the contents use row--fluid without the wrapper class.

Fluid Columns

Add row--fluid class to the row and make all the columns col--fluid, space will be equally divided among the columns.

  <div class="col--fluid">
    ...
  </div>
</div>

If there is an absolute column span inside the row--fluid, then the remaining space will be equally divide amoung col--fluid classes.

Responsive

BreakpointsInclude Frame Examples

You could make breakpoints to change the layout by targeting specific screen sizes. Add col-[*]-sm / col-[*]-md / col-[*]-lg / col-[*]-xl to target individual screen sizes.

[*] in this context indicates the col-span you need at the specific breakpoint.

Column 1
Column 2
Column 3
                  <div class="row">
  <div class="col-4-lg col-6-md col-12-sm">
    Column 1
  </div>
  <div class="col-4-xl col-6-md ">
    Column 2
  </div>
  <div class="col-4">
    Column 3
  </div>
</div>

12 : 8 : 4 LayoutInclude a link to 12:8:4 system

You could use the 12 : 8 : 4 column layout without specifying any breakpoints, but only making the columns responsive using col-[*]--res.

At lg and xl -> 12 col system, at md -> 8 col system and sm and xs -> 4 col system

  <div class="col-6--res">

By Numbers

You could use col--order-[*] class to order the columns by indicating their row position as a number.

[*] in this content indicates a number. The numbering goes from 0 - 12, where 0 indicates the first column and 12 indicating the last. You also have to notice if the rest of the columns are unordered and only one column is ordered with any other number except 0 or 12 then that column will be the last.

Center

You could use row--justify-center class to justify the columns to the center of the row.

</div>
